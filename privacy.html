Privacy Policy_

Última atualização: Março de 2026

O Kest foi desenvolvido com o princípio de Privacy First. Esta Política de Privacidade explica como as informações são tratadas dentro do aplicativo.

1. Informações Coletadas

O Kest não coleta, armazena ou transmite dados pessoais identificáveis para servidores externos. Todas as informações inseridas pelo usuário permanecem exclusivamente no dispositivo.

2. Armazenamento Local

Os dados inseridos no aplicativo (assinaturas, valores, datas e categorias) são armazenados localmente no dispositivo utilizando banco de dados interno baseado em SQLite. Nenhuma informação é enviada para servidores externos.

3. Permissões do Aplicativo

O aplicativo pode solicitar permissões limitadas do sistema operacional apenas para funcionalidades essenciais, como autenticação biométrica ou notificações locais. Essas permissões são utilizadas exclusivamente dentro do dispositivo.

4. Autenticação Biométrica

Caso o usuário utilize autenticação biométrica (Face ID ou impressão digital), essa verificação é processada inteiramente pelo sistema operacional do dispositivo. O Kest não coleta, armazena ou possui acesso aos dados biométricos do usuário.

5. Compras dentro do Aplicativo

O Kest pode oferecer recursos premium através de compras dentro do aplicativo. Todos os pagamentos são processados pelas plataformas oficiais de distribuição, como Google Play ou Apple App Store. O Kest não tem acesso a informações de pagamento ou dados financeiros do usuário.

6. Segurança

Como os dados são armazenados apenas no dispositivo, a segurança das informações depende principalmente da proteção do próprio aparelho. Recomendamos o uso de senha, biometria ou outros mecanismos de proteção disponíveis no sistema.

7. Privacidade de Crianças

O Kest não é direcionado a crianças menores de 13 anos. O aplicativo não coleta dados pessoais de usuários dessa faixa etária.

8. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir melhorias no aplicativo ou alterações legais. A data da última atualização será sempre exibida nesta página.
