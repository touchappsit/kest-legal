Privacy Policy_

Última atualização: Março de 2026

O Kest foi projetado com o princípio de Privacidade em Primeiro Lugar. Abaixo detalhamos como suas informações são tratadas.

1. Armazenamento Local

Todos os dados inseridos no aplicativo (assinaturas, valores, datas) são armazenados exclusivamente no seu dispositivo através de um banco de dados interno (SQLite). Não possuímos servidores de banco de dados e não temos acesso às suas informações.

2. Coleta de Dados

O Kest não coleta, transmite ou compartilha dados pessoais identidificáveis. Não utilizamos ferramentas de analytics ou rastreamento de comportamento.

3. Segurança e Biometria

O acesso biométrico (FaceID/Digital) é processado inteiramente pelas APIs do sistema operacional. O Kest apenas recebe a confirmação de autenticação e não armazena seus dados biométricos.

4. Exclusão de Dados

Ao desinstalar o aplicativo, todos os dados são removidos permanentemente pelo sistema operacional. O usuário é o único responsável pelo backup através da função de exportação.
